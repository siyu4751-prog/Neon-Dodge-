# Neon-Dodge-
# Neon Dodge 彩色躲避小游戏

这是一个使用 **C++ 控制台** 实现的小游戏，包含彩色界面、键盘控制、金币收集、障碍躲避、生命值系统、关卡速度提升和音效提示。项目不依赖第三方库，适合放在 GitHub 作为 C++ 入门项目展示。

# 项目亮点

- 使用 C++ 实现完整游戏循环
- 控制台彩色输出，角色、金币、障碍物颜色不同
- 使用 `Beep()` 实现金币、受伤、游戏结束音效
- 支持键盘实时操作
- 支持生命值、得分、等级、最高分保存
- 代码结构清晰，适合课程设计或 GitHub 展示

# 游戏玩法

玩家控制底部的角色 `A` 左右移动：

- `A` / `←`：向左移动
- `D` / `→`：向右移动
- `Q`：退出游戏

游戏元素：

- `$`：金币，接到后加分并播放提示音
- `X`：障碍物，碰到后扣生命值并播放碰撞音
- `+`：能量道具，接到后加生命值

分数越高，游戏速度越快。

# 运行环境

推荐使用 Windows 系统运行，因为音效使用了 Windows API 中的 `Beep()`。在 Linux/macOS 下也可以编译运行，但声音会退化为终端提示音。

#编译运行

## 方法一：使用 g++

在项目根目录打开终端，输入：

```bash
g++ src/main.cpp -o NeonDodge.exe
./NeonDodge.exe
```

Windows PowerShell 中可以这样运行：

```bash
.\NeonDodge.exe
```

##方法二：使用 Dev-C++ / CodeBlocks / Visual Studio

1. 打开 `src/main.cpp`
2. 编译运行
3. 进入游戏界面后按 Enter 开始
